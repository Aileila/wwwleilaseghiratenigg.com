# Portfolio — Leila Seghirate-Nigg

Portfolio professionnel mettant en avant des projets de pédagogie digitale, d'IA, et d'automatisation.

## 📁 Structure du projet

```
portfolio/
├── index.html                 # Page d'accueil (portfolio)
├── a-propos.html              # À propos & biographie
├── frameworks.html            # Frameworks pédagogiques
├── methodologie.html          # Méthodologies d'enseignement
├── offre.html                 # Offre de services
├── projets/                   # Case studies (pages de projets détaillés)
├── assets/                    # Ressources statiques
│   └── images/
│       ├── logos/             # Logos et icônes
│       ├── heroes/            # Images hero/bannières
│       ├── projects/          # Visuels des projets
│       ├── team/              # Photos de l'équipe
│       └── other/             # Autres images
├── tools/                     # Scripts utilitaires
├── .gitignore                 # Fichiers à ignorer dans Git
└── README.md                  # Ce fichier
```

## 🚀 Déploiement

### Sur Netlify (recommandé)

1. **Via GitHub** (recommandé pour les mises à jour auto)
   - Poussez le code sur GitHub: `git push`
   - Connectez votre repo GitHub à Netlify
   - Chaque push déclenche un déploiement automatique

2. **Via Drag & Drop** (rapide, mais manuel)
   - Allez sur [netlify.com](https://netlify.com)
   - Glissez-déposez le dossier du projet
   - Netlify génère une URL automatiquement

### Caractéristiques
