L'enjeu : faire ressentir avant de faire comprendre

Les formations sur les discriminations de genre en entreprise souffrent d'un problème structurel : elles expliquent sans faire vivre. Les apprenants sortent informés mais non transformés. Le plafond de verre reste un concept abstrait, pas une réalité incarnée.

L'hypothèse de départ : si on pouvait d'abord déclencher l'identification émotionnelle — faire que les participant·es voient le mécanisme dans une histoire avant de le nommer —, la compréhension conceptuelle suivrait avec une résistance bien moindre.

D'où l'idée d'une série narrative immersive : 5 personnages féminins construits comme des archétypes reconnaissables, 5 situations réelles d'invisibilisation, 5 modules permettant à chaque apprenante de se voir dans l'une d'entre elles.

L'IA comme co-auteure — pas comme outil

Ce projet est une démonstration concrète de ce que l'IA rend possible quand on l'engage comme partenaire de création, pas comme assistant de rédaction. Chaque dimension du projet a été co-construite avec des modèles de langage.

Les personnages ont été développés à partir de briefs de situations réelles, affinés par itérations successives pour atteindre la densité émotionnelle nécessaire. Les visuels de portraits ont été générés via IA d'image avec des prompts cinématographiques précis. Le code HTML/CSS/JS a été produit par vibe coding, testé et ajusté en dialogue.

La pédagogie elle-même — le choix des 5 mécanismes, la structure des quiz, les questions de débriefing — a émergé d'un travail de co-conception avec l'IA nourri des frameworks de sciences de l'apprentissage.

Ce que ce projet démontre

Plafond de Verre est une preuve de concept en vraie grandeur : il est possible de produire une expérience pédagogique de niveau professionnel — qualité de production, profondeur narrative, cohérence technique — en solo, avec l'IA comme co-auteure.

C'est aussi une réponse concrète à la question que posent tous les responsables formation : comment rendre les sujets RH sensibles assez engageants pour que les apprentissages restent ? La réponse est dans la forme autant que dans le fond.
